Token

Character, like alpha, digit and symbol, is used to composite tokens.

Character, like space, is used to separate tokens and prevent ambiguity.

// try to indentify all the tokens in this source code

#include <litan.h>

int main() {

    // get a float value from command line
    put_string("input radius: ");
    float radius;
    radius = get_float();

    // declare that the type of pi is float, and assign 3.14 to pi
    float pi = 3.14;

    // put the caculated float value to command line
    put_string("the area of given radius is: ");
    put_float(pi * radius * radius);

    return 0;
}

Type System in C

Every literal and variable has its type.

char literal starts and ends with quotation mark '.

string literal starts and ends with double quot ".

int literal and float literal start with digit.

true and false are bool literal.

Data and Type

Whatever we define literal, variable, constant, function or even operator as, they are all data. All data has its type in C.

Data and type are different, while structural token serves for grammar.

Data can either be constant or not.

If a variable can be assigned to a value, the variable is modifiable, writable. Otherwise, it is unmodifiable, read-only or constant.

Variables like main, put_string are constant, while variable like pi is modifiable. Hence, expression like pi = 2.71828 is legal while main = 3 are not.

All literals are constant.

Identifier and Keyword

Identifier is token that neither starts with digit, nor contains symbol except underscore _.

Keyword includes those identifiers of data that can are predefined in C language itself, not in standard library, third party library or ourselves.

Keyword Identifier but not Keyword
void true return if pi radius put_string pause

A list of keywords is predefined in C and can not be identifiers.

An Example

The following quotation block analysed the C source code above.

Every variable defined has a type and a value.

Executing the float radius; variable definition statement defines a variable with name pi, type float and value undefined.

Executing the radius = get_float(); assignment expression statement, replaces the get_float() expression with the value inputted, and sets the value of radius to it.

The float pi = 3.14; statement is a variable initialization statement. This statement can be viewed as a combination of the float pi; definition statement and pi = 3.14; assignment expression statement.

With the given context above, in the pi * radius * radius expression, pi is replaced with 3.14 and radius is replace with the value inputted.

The value of the expression is finally put on screen.
