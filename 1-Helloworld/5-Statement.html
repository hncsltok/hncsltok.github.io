Statement

The following table is about the classification, grammar and examples of statements.

Statements Grammar Example
Expression Statement <expression>; 1 + 2 * 3;
put_string("input radius");
radius = get_float();
Structural Statement return <expression>;
if (<expression>) { <statements> }
while (<expression>) { <statements> }
return 0;
Declaration extern <type> <expression>;
Definition <type> <expression>; float radius;
Initialization <type> <expression> = <expression>;
<type> <expression> = { <expressions> };
<type> <expression> { <statements> } 		float pi = 3.14
int main() { ... }

Effect of Statement

The following table is about effect of statement.

Statement Effect
Expression Statement calculate expression, assign value to variable, get data from user, or put data to user
Structural Statement change the order how statements are executed.
Declaration tell the type of a variable
Definition tell the type of a variable, and allocate storage for it
Initialization tell the type of a variable, and set the value of it to a certain one

The necessarity of these statements is explained at the end of Chapter-2.
