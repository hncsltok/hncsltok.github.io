Function

Any function variable, like f, is an function pointer. By using get-content operator *, *f is the content of function pointer.

A function call expression can be something like f(x) or (*f)(x).

Function is called with the content of function pointer like (*f)(x). f(x) is a grammar sugar for (*f)(x).

Function Initialization Statement

<type> <expression> = <expression>; is the grammar for simple type initialization statement.

<type> <expression> = { <expressions> }; is the grammar for array initialization statement.

Function initialization statement may be something like <type> <expression> = { <statements> };. But this is not supported and the following grammar should be used.

<type> <expression> { <statements> }

Notice that = operator and ; token is omitted.

To initialize an function with another function, still using <type> <expression> = <expression>;.
